# Projects

## Job Search Action Center

- **Purpose:** automate daily job discovery, scoring, routing, application drafting, and networking follow-up.
- **Mobile coding path:** open the GitHub repository in Codespaces from a phone.
- **Setup command:** `python -m pip install -e ".[dev]"`
- **Test command:** `python -m pytest -q`
- **Run command:** `python -m runner`
- **Important local files:** `.env`, `resume/`, `secrets/`, and exported data are intentionally ignored by git.
